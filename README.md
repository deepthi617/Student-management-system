# Smart Student Management System
A full-stack Student Management System developed using React.js, Spring Boot, REST APIs, and MySQL. The application allows users to manage student records efficiently through a simple and responsive web interface.

## 🚀 Features
- Add Student Records
- View Student Details
- Update Student Information
- Delete Student Records
- RESTful API Integration
- Responsive User Interface

## 🛠️ Technologies Used

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript (ES6+)

### Backend
- Spring Boot
- REST APIs
- Java

### Database
- MySQL

### Tools
- Git
- GitHub
- VS Code
